### Main dev commands
1.  `pnpm install`
2.  `pnpm start` (for react-app) and
3.  `pnpm watch-css` for watching sass files.

### Dev scripts
#### On Windows
If `pnpm` is installed run 'start.bat'

#### On Ubuntu/gnome-terminal
If `pnpm` is installed `sh start.sh`
